# Adjustable-Voltage-Regulator
A voltage regulator with adjustable output that includes two power pin headers for input and output, over-voltage and reverse polarity protection, 5V voltage regulators, LED Indicators for Power and over voltage, and a adjustable voltage regulator from the AMS1117-ADJ alongside a trimmer Potentiometer.
